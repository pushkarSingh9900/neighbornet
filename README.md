# NeighborNet
NeighborNet is a student-driven housing review platform where verified students can share experiences about houses they have lived in. The goal of the platform is to help students make better housing decisions by learning from the experiences of other students.

## Problem
Students often rely on rental ads or word-of-mouth when searching for housing. Many important issues such as poor heating, mold, pests, or safety concerns are discovered only after moving in. Currently, there is no trusted student-focused platform to share honest housing experiences.

## Solution
NeighborNet allows students to:
- Add houses they have lived in
- Write reviews about their experiences
- Report serious housing issues
- Communicate with other students through in-app messaging
An admin moderation system ensures reliability and prevents misuse of the platform.

## Features
- Student authentication using university email
- Property listing by area
- Housing reviews and ratings
- Issue reporting system
- In-app student messaging
- Admin moderation dashboard

## System Architecture
NeighborNet follows a client-server architecture:
- **Frontend:** React / Next.js
- **Backend:** Node.js + Express
- **Database:** MongoDB (NoSQL)
- **Storage:** Image storage for issue evidence

## Database Entities
The main entities in the system include:
- User
- Property
- Review
- IssueReport
- Message
- Flag

## Future Enhancements
Planned improvements include:
- AI-based housing recommendation system
- Chatbot to assist students with housing questions
- Advanced analytics on housing conditions
- Expansion to other universities and cities
